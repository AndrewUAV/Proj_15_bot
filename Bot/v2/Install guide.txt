Iнструкція з встановлення пакета за допомогою `pip install -e . :

1. Відкрийте термінал або командний рядок на вашому комп'ютері в  цій папці Proj_15_bot\cd Bot\1.1
                                                                або перейдіть у цю папку командою cd

2. У вікні терміналу виконайте наступну команду: pip install -e .

3. Після завершення встановлення ви можете запустити додаток командою bot_pg15 та bot_pg15 ui

або перевірити, чи правильно встановлений пакет, запустивши команду: pip list

видалити пакет можно командою pip uninstall our_bot_pg15
