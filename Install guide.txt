Iнструкція з встановлення пакета за допомогою `pip install -e .`:

1. Відкрийте термінал або командний рядок на вашому комп'ютері.

2. Перш ніж почати, переконайтеся, що на вашому комп'ютері встановлено Python та pip. Ви можете перевірити їх, виконавши команди `python --version` і `pip --version` у вашому терміналі або командному рядку.

3. Встановіть пакет prompt_toolkit за допомогою команди pip install prompt_toolkit

4. Використовуйте команду `cd` для переходу до папки, де знаходиться репозиторій з вашим пакетом.

5. У вікні терміналу виконайте наступну команду: pip install -e

6. Після завершення встановлення ви можете перевірити, чи правильно встановлений пакет, запустивши команду: pip list

7. Тепер ви можете імпортувати та використовувати ваш пакет у вашому Python коді, як будь-який інший встановлений пакет.

Таким чином, ви успішно встановили свій пакет у режимі редагування за допомогою `pip install -e .`. Тепер ви можете вносити зміни до його вихідного коду та використовувати їх безпосередньо у вашому коді.
